#各类快捷方式
+ F1显示当前程序或者windows的帮助内容  
+ F2当你选中一个文件时，为“重命名”  
+ F3在桌面上时打开“查找：所有文件”对话框  
+ F10或ALT激活当前程序的菜单栏  
+ Ctrl+N新建一个文件  
+ Ctrl+O打开“打开文件”对话框  
+ Ctrl+P打开“打印”对话框  
+ Ctrl+S保存  
+ Ctrl+X剪切
+ Ctrl+C复制
+ Ctrl+V黏贴
+ Ctrl+Z撤销
+ Ctrl+V黏贴
+ Ctrl+V黏贴
+ Windows+M最小化所有窗口
+ Windows+Ctrl+M恢复至最小化窗口之前
+ Windows+E打开资源管理器
+ Windows+R打开“运行”对话框
+ Windows+Break打开“系统属性”
+ Alt+f4关闭当前应用程序
+ Alt+TAB切换当前程序
+ Alt+ESC切换当前程序
